## 1. 手册怎么用

本手册按“你要做什么事”来写，不讲代码，不讲架构。  
每一节都按以下格式：

1. 你会看到什么界面  
2. 你要点哪里  
3. 成功后会看到什么结果  
4. 常见失败怎么处理  

文中带有 `【截图】` 的位置，请配套放入真实 App 截图（见文末“截图补齐说明”）。

## 2. 统一账号说明（所有 App 通用）

Verlu 全家应用（Sync / Talk / Doctor / CnChess / Music / Cloud）共用同一个账号。

- 在任意一个 App 注册后，可用同一邮箱密码登录其它 App
- 忘记密码可在登录页走“重置密码”
- 首次登录后建议先完善昵称和头像，便于好友识别

## 3. Sync：设备数据与天气

### 3.1 你能做什么

- 查看手机电量、温度、屏幕使用时长
- 查看当前位置天气
- 扫码授权其他设备登录

### 3.2 快速上手（第一次打开）

1. 打开 Sync，先登录账号  
2. 进入首页后依次查看底部 4 个功能页：电量 / 温度 / 屏幕时长 / 天气  
3. 按提示授权权限（定位、使用情况访问等）

`【截图】Sync 首页底部导航`  
![Sync-首页导航](./user-guide/assets/sync/01-home-tabs.png)

### 3.3 查看天气

1. 进入“天气”页  
2. 首次会提示定位授权，点“允许”  
3. 页面显示当前天气和未来预报

`【截图】天气页+定位授权弹窗`  
![Sync-天气页](./user-guide/assets/sync/02-weather-location.png)

### 3.4 扫码授权登录（给 Cloud 桌面用）

1. 在桌面 Cloud 打开登录二维码  
2. 手机 Sync 进入扫码页  
3. 对准二维码，确认授权  
4. 桌面端自动登录成功

`【截图】扫码页`  
![Sync-扫码授权](./user-guide/assets/sync/03-sso-scan.png)

## 4. Talk：聊天与好友

### 4.1 你能做什么

- 通过邮箱或二维码添加好友
- 查看“新的朋友”，处理申请
- 进入聊天，发送消息

### 4.2 添加好友（邮箱）

1. 进入“联系人”页  
2. 点击“添加好友”  
3. 输入对方邮箱并搜索  
4. 点击“发送申请”

`【截图】添加好友弹窗`  
![Talk-添加好友](./user-guide/assets/talk/01-add-friend-by-email.png)

### 4.3 处理好友申请（接收方）

1. 进入“联系人”→“新的朋友”  
2. 找到对方申请  
3. 点击“同意”或“拒绝”

`【截图】新的朋友列表`  
![Talk-新的朋友](./user-guide/assets/talk/02-new-friends.png)

### 4.4 开始聊天

1. 进入“会话”页  
2. 点击好友会话  
3. 输入内容发送  
4. 对方在线会即时收到

`【截图】聊天页`  
![Talk-聊天界面](./user-guide/assets/talk/03-chat-room.png)

### 4.5 扫码加好友

1. 对方在个人页展示二维码  
2. 你打开“扫码加好友”  
3. 扫码后确认发送申请

`【截图】扫码加好友`  
![Talk-扫码加好友](./user-guide/assets/talk/04-qr-add-friend.png)

## 6. CnChess：中国象棋对战

### 6.1 你能做什么

- 邀请好友下棋
- 实时对战（含棋钟）
- 提议和棋、认输、复盘历史

### 6.2 开局前准备

1. 确保你和对方在 Talk 已是好友  
2. 双方打开 CnChess  
3. 在好友列表找到对方

`【截图】好友列表页`  
![CnChess-好友列表](./user-guide/assets/cnchess/01-friends-list.png)

### 6.3 发起并接受邀请

1. 你点击“邀请对局”  
2. 对方收到弹窗  
3. 对方点“接受”  
4. 双方自动进入棋盘

`【截图】邀请弹窗`  
![CnChess-邀请弹窗](./user-guide/assets/cnchess/02-invite-dialog.png)

### 6.4 对局操作

1. 点击己方棋子  
2. 再点击目标位置走子  
3. 轮到对方时不可操作  
4. 可在对局内提议和棋或认输

`【截图】棋盘对局页`  
![CnChess-对局页](./user-guide/assets/cnchess/03-game-board.png)

### 6.5 查看历史与回放

1. 进入“对局历史”  
2. 点开一局历史  
3. 进入回放模式逐步查看

`【截图】历史列表+回放`  
![CnChess-历史回放](./user-guide/assets/cnchess/04-history-replay.png)

### 9.1 登录后台

3. 填写 `versionCode` / `versionName` / 更新说明

## 10. 常见问题（用户视角）

### Q1：登录成功后又回到登录页
- 检查网络是否可用  
- 关闭 App 后重开  
- 若仍失败，请使用“忘记密码”重置

### Q2：Talk 添加好友后，对方看不到
- 让对方打开“联系人 → 新的朋友”  
- 下拉刷新一次  
- 确认你输入的是对方绑定邮箱

### Q3：CnChess 邀请发出后对方没弹窗
- 确认双方都是好友  
- 对方需保持 CnChess 在前台或刚切回前台  
- 让对方重进首页

### Q4：Cloud 上传很慢
- 切换更稳定网络（Wi-Fi）  
- 大文件等待分片上传完成  
- 不要在系统里强杀 App
